Guia rápido

Como rodar localmente

1) Subir a API (porta 8080)

python app.py

2) Iniciar simulador IoT

java -jar simulator-sensores-iot.jar

3) Abrir o dashboard

http://localhost:8080/

O simulador envia leituras JSON para /api/sensor/data. A página se atualiza automaticamente.
